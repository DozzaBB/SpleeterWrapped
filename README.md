# SpleeterWrapped
wrapped nicer version of spleeter with youtube-dl included for speedy filing.


acquire via: 
* install python 3.10
* `python -m venv venv`
* `venv/Scripts/Activate`
* `pip install -r requirements.txt`
* install python 3.10
* `python -m venv venv`
* `venv/Scripts/Activate`
* `pip install -r requirements.txt`
